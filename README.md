# Задание на программирование (Разработка Web-приложения на Django)
## Как запустить сайт
1. Скачать этот репозиторий или клонировать его
2. Перейти в папку репозитория в консоли при помощи `cd`
3. Выполнить установку `pip install -r requirements.txt`
4. Запустить сервер `python manage.py runserver --insecure`
5. Открыть сайт по [ссылке](http://127.0.0.1:8000/)

Данный раздел открывает словарь, который содержит слова на английском языке и их перевод на русский язык

### Редактировать
В данном разделе можно добавить новое слово в словарь, изменить перевод существующего или удалить слово из словаря
<img width="1389" alt="image" src="https://github.com/denismox/PythonSite/assets/84687793/c67041cb-9fa8-4efc-91a2-227f0619bd3a">

В данном разделе можно проверь свои знания. Пользователю будет предложено 6 карточек со случайными словами из словаря для которых необходимо написать перевод

После проверки пользователь будет пернаправлен на страницу проверки, где будут показаны результаты в виде таблицы, а также предложено пройти тест заного, добавить новые слова в словарь, посмотреть словарик или вернуться на главную страницу
